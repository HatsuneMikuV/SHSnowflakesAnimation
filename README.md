# SHSnowflakesAnimation

### SHAnimation

- 雪花飘落动画---帧动画系列，雪花立体反转，可改变x,y,z三个不同方向反转效果__可自行拓展，雪花路线由贝塞尔曲线绘制而成__可自行更改路线

- 粒子动画---layer动画，可自动定时触发，也可按照用户手指点触发

- 雷达/咻咻---layer动画，可设定大小范围

- 抖动动画---layer动画，可自行定制方向、时长、幅度
